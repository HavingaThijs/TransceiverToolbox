  adi.FMComms4.Tx Transmit data from the FMComms4 evaluation platform
    The adi.FMComms4.Tx System object is a signal source that can 
    send complex data to the FMComms4.
 
    tx = adi.FMComms4.Tx;
    tx = adi.FMComms4.Tx('uri','192.168.2.1');
