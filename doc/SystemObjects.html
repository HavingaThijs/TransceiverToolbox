ADI BSP System Objects For Radio I/O

All ADI streaming interfaces share the same general API. Simply insert you specific part of development board:
rx = adi.<part_or_board_name>.Rx;
rx.uri = '<uri of board>';
receiveData = rx();
tx = adi.<part_or_board_name>.Tx;
tx.uri = '<uri of board>';
tx(receiveData);
Example:
rx = adi.AD9361.Rx
rx =
adi.AD9361.Rx with properties:
CenterFrequency: 2.4000e+09
SamplingRate: 3000000
RFBandwidth: 3000000
GainControlModeChannel0: 'slow_attack'
EnableQuadratureTracking: true
EnableRFDCTracking: true
EnableBasebandDCTracking: true
SamplesPerFrame: 32768
channelCount: 2
uri: 'ip:10.0.0.200'
rx.uri = 'ip:192.168.2.1';
rx.CenterFrequency = 1e9;
data = rx();
