  adi.ADRV9009.Tx Transmit data from the ADRV9009 transceiver
    The adi.ADRV9009.Tx System object is a signal sink that can tranmsit
    complex data from the ADRV9009.

    tx = adi.ADRV9009.Tx('uri','192.168.2.1');
